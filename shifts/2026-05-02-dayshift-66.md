Direction taxonomy (`JavaErrorsGoCorrect` / `JavaWrongRowsGoCorrect` / `BothErrorMessagesDrift` / `JavaSucceedsGoRejects`) is grep-friendly for upstream-bug accounting. Harness asserts Go's behaviour but NOT Java's; if Java upstream fixes the bug, the assertion fires `upstream may be fixed; revisit annotation`. Migrated 7 entries (#42 / #44×2 / #48 / #52 / #53 / #64) into the corpus and deleted 5 corresponding Go-only sentinel tests.
