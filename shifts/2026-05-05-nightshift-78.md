1. **Derived table Cascades translation** — the embedded path materializes derived tables as CTEs; the Cascades path needs equivalent handling. Affects ~10 scenarios.
2. **Subquery support** — EXISTS, scalar subqueries, correlated subqueries. Needs SelectExpression + subquery translation. Affects ~8 scenarios.
3. **#90 strictlySorted** — reverse scan with equality filter + DESC produces wrong order. Affects ~3 scenarios.
4. **sortOnly column stripping** — ORDER BY SUM(v)*2 includes SUM(V) in output. Needs post-sort projection that only strips when there are actual sortOnly ORDER BY aggregates (not HAVING-harvested hidden ones). Attempted but reverted due to regression.
5. **3-way JOIN column resolution** — NLJ row merge overwrites overlapping column names. Inner `NAME` overwrites outer `NAME`. Fix needs careful handling of ambiguity.
6. **LEFT/RIGHT OUTER JOIN** — needs outer join implementation in Cascades.
