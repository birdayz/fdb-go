### 4. CI infrastructure

- **Concurrency groups**: Added to CI + Claude Code workflows. Stale intermediate runs on the same branch are auto-cancelled. Prevents queue congestion on the self-hosted runner.
- **Claude Code bun fix**: Debugged `bun: command not found` in `claude-code-action` post-step. Root cause: `~/.bun/bin/bun` cache went cold after the runner's disk was stressed by binding stress Docker containers. Fixed by adding `oven-sh/setup-bun@v2` step before `claude-code-action` (pushed to master).
